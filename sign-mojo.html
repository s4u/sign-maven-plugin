sign:sign

Full name:

org.simplify4u.plugins:sign-maven-plugin:0.2.0:sign

Description:

Creates OpenPGP signatures for all of the project's artifacts.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 0.1.0.
  • Binds by default to the lifecycle phase: verify.

Optional Parameters

Name Type Since Description
<keyFile> File 0.1.0

File with private key used for signing.

This value can be delivered by environment variable SIGN_KEY. Environment variable must contain private key content.

Key can by created and exported by:

     gpg --armor --export-secret-keys

Default value is: ${user.home}/.m2/sign-key.asc.
User property is: sign.keyFile.
<keyId> String 0.1.0

keyId used for signing. If not provided first key from keyFile will be taken.

This value can be delivered by environment variable SIGN_KEY_ID.


User property is: sign.keyId.
<keyPass> String 0.1.0

passphrase to decrypt private signing key.

Provided key can be stored in plain text, in this case keyPass can be empty.

This value can be delivered by environment variable SIGN_KEY_PASS.


User property is: sign.keyPass.
<skip> boolean 0.1.0 Skip the execution of plugin.
Default value is: false.
User property is: sign.skip.
<skipNoKey> boolean 0.1.0 By default execution is skipped if private key is missing.

When set to false and private key is missing error will be reported for current Maven session.


Default value is: true.
User property is: sign.skipNoKey.
